MCS MANAGER
Version 8 | Yelin Plan

我们用最简单上手的方式，来构建整个应用程序，绝大部分的开发者可以轻而易举的进行修改，以及依照协议开发使用。你会发现，这可能是一非常适合你的 Minecraft 服务器管理工具，支持绝大部分的主流服务端。

并且，我们将尽可能的降低扩展的学习成本来帮助你更好的扩展与自定义，让一个完全属于你的Minecraft服务器管理器出现。


运行环境需求

Linux: Node.js >= 8.0

Windows: 自带移动绿色环境

Other: 阅读 README.md

浏览器兼容性

ECMAScript 5 标准

IE 10+ Chrome Firefox Safari

等现代主流浏览器

轻量与简单

无需依赖额外其他数据库

无需配置过多的设置选项

安装好环境即可直接运行

方便部署

Linux 几条命令即可部署

Windows 双击即可运行

可适用于任何支持 Node 环境机器
